    AWS Data Exchange

    The AWS Data Exchange is a service that makes it easy to find and use data for cloud computing. AWS has added a clearinghouse for Data related to COVID-19 available for Research & Development.

    You can access data like the Unacast social-distancing scoreboard data through AWS[1], often for free. Systems like the Coronavirus Tracker from Domo are powered by data from the exchange.
